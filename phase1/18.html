1️⃣ What is React?

📌 Purpose & Usage

React is a JavaScript library for building user interfaces, created by Meta (Facebook). It helps developers build fast, interactive, and reusable UI components using a declarative approach.

Where it's used: Facebook, Instagram, Netflix, WhatsApp Web, Airbnb, UberEats — thousands of modern web applications.

⚡ React at a Glance

function Welcome() {
return <h1>Hello from React!</h1>;
}
// Component is reusable, composable, and reactive
✅ React uses a component-based architecture.
✅ Each component returns JSX (HTML-like syntax).
✅ React updates only the changed parts (Virtual DOM).
✅ Makes building complex UIs much easier.
2️⃣ Why React? (Traditional JS vs React)

📌 The Problem React Solves

Traditional JavaScript requires manual DOM manipulation (document.getElementById, innerHTML) which becomes messy for large apps. React uses a declarative approach — you describe what the UI should look like, and React handles updates efficiently.

// Traditional JS
const el = document.getElementById("counter");
let count = 0;
el.innerHTML = count;
// React — declarative
function Counter() {
const [count, setCount] = useState(0);
return <button onClick={() => setCount(count+1)}>{count}</button>;
}
✅ React = Declarative (what you want) vs Imperative (how to do it)
✅ Virtual DOM makes updates fast
✅ Components are reusable and maintainable
✅ React is used by industry giants for a reason!
3️⃣ SPA (Single Page Application)

📌 What is SPA?

Single Page Application loads a single HTML page and dynamically updates content without full page reloads. React apps are SPAs by default, providing smooth user experience like Gmail, YouTube, Facebook.

Traditional: Page 1 → Reload → Page 2 → Reload → Page 3
SPA: Page 1 → Content updates → Content updates (No reload) 🚀
4️⃣ Create React Project using Vite (Fast & Modern)

📌 Setup Steps

Vite is a modern build tool that creates React projects quickly with fast hot module replacement.

# Step 1: Create project
npm create vite@latest my-react-app

# Step 2: Select React + JavaScript
# Step 3: Install dependencies
cd my-react-app
npm install

# Step 4: Run dev server
npm run dev
✅ After running npm run dev, open http://localhost:5173
✅ You'll see your first React app!
✅ Vite provides instant hot reloading when you edit files.

6️⃣ JSX (JavaScript XML) – HTML inside JavaScript

📌 JSX Explained

JSX allows you to write HTML-like syntax directly in JavaScript. It gets compiled to React.createElement calls. JSX makes React components readable and intuitive.

function Greeting() {
let name = "Kalyan";
return <h1>Hello {name}!</h1>; // Embed JS with { }
}
📦 JSX Output Simulation:

Hello Kalyan!

✅ JSX allows HTML + JavaScript together

JSX is transformed to React.createElement('h1', null, 'Hello Kalyan!')
7️⃣ React Components (Reusable UI Blocks)

📌 Component-Based Architecture

Components are independent, reusable pieces of UI. Think of them as LEGO blocks — you can combine them to build complex interfaces.

function Header() {
return <header>My Website</header>;
}
function Footer() {
return <footer>© 2024</footer>;
}
function App() {
return (<><Header /><Footer /></>);
}
📌 Header Component
📄 Main Content Area
© 2024 Footer Component

9️⃣ Dynamic Data & Expressions ({})

✅ Use { } to embed any JavaScript expression in JSX!
🔟 React Events (onClick, onChange)

📌 Events in React

React events are named using camelCase (onClick instead of onclick). Event handlers are functions passed as references.

function App() {
const handleClick = () => {
alert("Button clicked in React!");
};
return <button onClick={handleClick}>Click Me</button>;
}
🖱️ React-style Event Button:
✅ onClick={handler} (camelCase, function reference)
📁 Mini Project: React Interactive Button
function App() {
const [message, setMessage] = React.useState("");
return (
<div>
<button onClick={() => setMessage("Welcome to React!")}>Greet</button>
<p>{message}</p>
</div>
);
}

✅ In real React, useState would manage this message state!
⚠️ Common Beginner Mistakes in React
  • ❌ Using class instead of className for CSS classes in JSX. Fix: className="title".
  • ❌ Not returning single parent element → use fragments <> </>.
  • ❌ Component names not starting with capital letter → function app() ❌, function App() ✅.
  • ❌ Forgetting to export component → export default App is required.
  • ❌ Calling functions in render instead of passing reference: onClick={handleClick()} ❌ vs onClick={handleClick} ✅.
