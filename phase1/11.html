3️⃣ Basic Array Methods (push, pop, shift, unshift)

➕➖ Modify Arrays

let arr = [1, 2];
arr.push(3); // [1,2,3]
arr.pop(); // [1,2]
arr.unshift(0); // [0,1,2]
arr.shift(); // [1,2]

4️⃣ forEach() — Loop Through Array

🔄 Iteration with forEach

let fruits = ["Apple", "Mango", "Orange"];
fruits.forEach((fruit, index) => {

});

5️⃣ map() — Transform Every Element

💡 map() creates a NEW array by applying function to each element

📐 Square numbers using map

let numbers = [1, 2, 3, 4];
let squared = numbers.map(n => n * n);
// squared = [1, 4, 9, 16]
🔢 map() transformation:
6️⃣ filter() — Select Matching Elements

🎯 Filter numbers greater than 30

let numbers = [10, 20, 30, 40, 50];
let filtered = numbers.filter(num => num > 30);
// [40, 50]
✅ filter() in action:
7️⃣ reduce() — Accumulate into Single Value

💰 Calculate total sum

let cart = [100, 200, 50];
let total = cart.reduce((sum, price) => sum + price, 0);
// total = 350

⚠️ Common Beginner Mistakes
  • ❌ Using map() without returning a value → results in undefined array. Always use return or arrow shorthand.
  • ❌ Confusing filter() (returns array) vs find() (returns single element).
  • ❌ Accessing index out of bounds → arr[10] when length is 3 returns undefined.
  • ❌ Mutating original array accidentally → use spread [...arr] to copy.
  • ❌ Forgetting that sort() mutates original array.
