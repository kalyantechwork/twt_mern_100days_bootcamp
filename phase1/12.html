1️⃣ What is an Object?
📚 Object stores data in key: value pairs — like a real-world entity (student, product, user).

📦 Creating an Object

let student = {
name: "Kalyan",
age: 25,
city: "Hyderabad"
};
📋 Object Output:
2️⃣ Accessing & Updating Object Values

🔍 Dot & Bracket Notation

let user = { name: "Priya", role: "Developer" };
console.log(user.name); // Dot notation
console.log(user["role"]); // Bracket notation
user.city = "Chennai"; // Add new property
delete user.role; // Remove property
🔄 Object Operations:
3️⃣ Nested Objects (Object inside Object)

🏠 Nested Structure

let user = {
name: "Kalyan",
address: {
city: "Hyderabad",
pincode: 500001
}
};
console.log(user.address.city); // Hyderabad

5️⃣ Object.keys() & Object.values()

🔑 Getting Keys and Values

let student = { name: "Kalyan", age: 25 };
Object.keys(student); // ["name", "age"]
Object.values(student); // ["Kalyan", 25]

6️⃣ Object Methods & this Keyword

📞 Method inside Object

let user = {
name: "Kalyan",
greet() {
return `Hello, ${this.name}!`;
}
};
user.greet(); // "Hello, Kalyan!"

7️⃣ Object Destructuring (Extract Values Easily)

📦 Destructuring Syntax

let student = { name: "Kalyan", age: 25, city: "Hyderabad" };
let { name, age } = student;
console.log(name, age); // Kalyan 25

// Renaming variables
let { name: studentName } = student;
📦 Destructuring in Action:
8️⃣ Array Destructuring (Unpack Arrays)

🎯 Array Destructuring

let fruits = ["Apple", "Mango", "Orange"];
let [first, second] = fruits;
console.log(first); // Apple
console.log(second); // Mango
📦 Array Destructuring:
9️⃣ Spread Operator (...) with Objects (Copy & Merge)

✨ Spread for Objects

let user = { name: "Kalyan" };
let updated = { ...user, city: "Hyderabad" };
// updated = { name: "Kalyan", city: "Hyderabad" }

⚠️ Common Beginner Mistakes
  • ❌ Using dot notation with variables: student["name"] not student[name] (unless variable holds the key).
  • ❌ Accessing undefined property: user.address.city when address is undefined → use optional chaining user?.address?.city.
  • ❌ Confusing arrays of objects vs object of arrays.
  • ❌ Destructuring undefined variables → provide default values: let { name = "Guest" } = user;
  • ❌ Forgetting commas between properties in objects.
