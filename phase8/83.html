1️⃣ What is CRUD?

CRUD stands for Create, Read, Update, Delete — the four basic operations for persistent storage. Every application uses CRUD!

Create (POST) → Read (GET) → Update (PUT/PATCH) → Delete (DELETE) → Complete Data Management
💡 Real-world examples: Adding a tweet (Create), viewing feed (Read), editing profile (Update), deleting a post (Delete). CRUD everywhere!

3️⃣ Backend CRUD API — Express Routes
// routes/products.js
const express = require('express');
const router = express.Router();

let products = [
{ id: 1, name: "Laptop", price: 999, inStock: true },
{ id: 2, name: "Phone", price: 499, inStock: true }
];

// CREATE - POST
router.post('/', (req, res) => {
const newProduct = { id: Date.now(), ...req.body };
products.push(newProduct);
res.status(201).json(newProduct);
});

// READ all - GET
router.get('/', (req, res) => {
res.json(products);
});

// READ single - GET by ID
router.get('/:id', (req, res) => {
const product = products.find(p => p.id === parseInt(req.params.id));
if (!product) return res.status(404).json({ error: "Product not found" });
res.json(product);
});

// UPDATE - PUT
router.put('/:id', (req, res) => {
const index = products.findIndex(p => p.id === parseInt(req.params.id));
if (index === -1) return res.status(404).json({ error: "Product not found" });
products[index] = { ...products[index], ...req.body };
res.json(products[index]);
});

// DELETE - DELETE
router.delete('/:id', (req, res) => {
const index = products.findIndex(p => p.id === parseInt(req.params.id));
if (index === -1) return res.status(404).json({ error: "Product not found" });
products.splice(index, 1);
res.status(204).send();
});

module.exports = router;
4️⃣ Frontend CRUD — Axios Service
// services/productService.js
import axios from 'axios';

const API_URL = 'http://localhost:5000/api/products';

export const productService = {
// CREATE
create: async (product) => {
const response = await axios.post(API_URL, product);
return response.data;
},
// READ (all)
getAll: async () => {
const response = await axios.get(API_URL);
return response.data;
},
// READ (single)
getById: async (id) => {
const response = await axios.get(`${API_URL}/${id}`);
return response.data;
},
// UPDATE
update: async (id, product) => {
const response = await axios.put(`${API_URL}/${id}`, product);
return response.data;
},
// DELETE
delete: async (id) => {
await axios.delete(`${API_URL}/${id}`);
}
};

};

const newProduct = await productService.create(form);
setProducts([...products, newProduct]);
setForm({ name: '', price: '' });
};

const handleUpdate = async () => {
const updated = await productService.update(editing.id, form);
setProducts(products.map(p => p.id === editing.id ? updated : p));
setEditing(null);
setForm({ name: '', price: '' });
};

const handleDelete = async (id) => {
await productService.delete(id);
setProducts(products.filter(p => p.id !== id));

⚠️ Common CRUD Mistakes
  • ❌ Not updating UI after Create/Update/Delete → state not refreshed
  • ❌ Using wrong HTTP method (GET for delete, POST for update)
  • ❌ Not handling loading states → UI freezes during API calls
  • ❌ Not validating data before sending to backend
  • ❌ Forgetting to import/use async/await properly
💼 Common Interview Questions

❓ What is CRUD?

Create, Read, Update, Delete — the four basic operations for data management.

❓ HTTP methods for CRUD?

POST (Create), GET (Read), PUT/PATCH (Update), DELETE (Delete).

❓ Difference between PUT and PATCH?

PUT replaces entire resource, PATCH partially updates.
