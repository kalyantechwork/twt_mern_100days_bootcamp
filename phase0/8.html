⚠️ Common Beginner Mistakes (Functions)
  • ❌ Forgetting to call function: myFunction vs myFunction()
  • ❌ Missing return: function without return returns undefined
  • ❌ Accessing local variable outside function → ReferenceError
  • ❌ Confusing parameters and arguments
  • ❌ Using arrow function with wrong syntax: const add = (a,b) => { a+b } → missing return or {} with return
