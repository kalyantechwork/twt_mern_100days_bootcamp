CRUD Operations – MongoDB Shell & Compass

Master all CRUD operations (Create, Read, Update, Delete) in MongoDB using both the Shell and Compass GUI. Learn to manipulate data like a pro!

Telugu WebTech · Complete MongoDB CRUD Tutorial
1. What is CRUD?

CRUD stands for Create, Read, Update, Delete – the four basic operations for any database. Every application you build will use these operations.

CREATE

Insert new documents
📝 InsertOne, InsertMany

READ

Query/find documents
🔍 Find, FindOne

UPDATE

Modify existing documents
✏️ UpdateOne, UpdateMany

DELETE

Remove documents
🗑️ DeleteOne, DeleteMany

3. CREATE – Inserting Data
📝 Insert One Document
// MongoDB Shell db.users.insertOne({ name: "Ramesh", email: "ramesh@example.com", age: 25, city: "Hyderabad", isActive: true, hobbies: ["coding", "reading"] }) // Response { acknowledged: true, insertedId: ObjectId("507f1f77bcf86cd799439011") }

// MongoDB Shell db.users.insertMany([ { name: "Priya", email: "priya@example.com", age: 22, city: "Mumbai", isActive: true, hobbies: ["dancing", "music"] }, { name: "Suresh", email: "suresh@example.com", age: 28, city: "Delhi", isActive: false, hobbies: ["gaming", "coding"] }, { name: "Anjali", email: "anjali@example.com", age: 30, city: "Bangalore", isActive: true, hobbies: ["yoga", "reading", "travel"] } ]) // Insert more sample data db.users.insertMany([ { name: "Vikram", email: "vikram@example.com", age: 35, city: "Chennai", isActive: true, hobbies: ["swimming", "chess"], orders: 5 }, { name: "Neha", email: "neha@example.com", age: 26, city: "Hyderabad", isActive: true, hobbies: ["painting", "photography"] }, { name: "Raj", email: "raj@example.com", age: 32, city: "Pune", isActive: false, hobbies: ["trekking", "gaming"] } ])
🖥️ In Compass
1. Select the "users" collection 2. Click "Add Data" → "Insert Document" 3. Paste JSON document 4. Click "Insert"
4. READ – Querying Data
📖 Find All Documents
// MongoDB Shell db.users.find() // Pretty print (formatted) db.users.find().pretty() // Get all documents as array db.users.find().toArray() // Count documents db.users.countDocuments()
📖 Find One Document
// Find first matching document db.users.findOne({ name: "Ramesh" }) // Find by _id db.users.findOne({ _id: ObjectId("507f1f77bcf86cd799439011") })
📖 Find with Filters
// Equal to db.users.find({ city: "Hyderabad" }) // Greater than db.users.find({ age: { $gt: 25 } }) // Greater than or equal db.users.find({ age: { $gte: 25 } }) // Less than db.users.find({ age: { $lt: 30 } }) // Not equal db.users.find({ city: { $ne: "Mumbai" } }) // Multiple conditions (AND) db.users.find({ city: "Hyderabad", isActive: true }) // OR condition db.users.find({ $or: [ { city: "Hyderabad" }, { city: "Bangalore" } ] }) // IN operator db.users.find({ city: { $in: ["Hyderabad", "Mumbai", "Delhi"] } }) // EXISTS operator db.users.find({ orders: { $exists: true } }) // Array contains value db.users.find({ hobbies: "coding" }) // Array contains multiple values db.users.find({ hobbies: { $all: ["coding", "reading"] } })
📖 Sorting, Limiting, and Projection
// Sort by age ascending (1) or descending (-1) db.users.find().sort({ age: 1 }) db.users.find().sort({ age: -1 }) // Limit results db.users.find().limit(3) // Skip (for pagination) db.users.find().skip(2).limit(3) // Projection (select specific fields) db.users.find({}, { name: 1, email: 1, _id: 0 }) // Chain methods db.users.find({ isActive: true }) .sort({ age: -1 }) .limit(5)
5. UPDATE – Modifying Data
✏️ Update One Document
// $set – Update specific fields db.users.updateOne( { name: "Ramesh" }, { $set: { age: 26, city: "Secunderabad" } } ) // $inc – Increment a number field db.users.updateOne( { name: "Ramesh" }, { $inc: { age: 1 } } ) // $push – Add to array db.users.updateOne( { name: "Priya" }, { $push: { hobbies: "singing" } } ) // $addToSet – Add to array (no duplicates) db.users.updateOne( { name: "Priya" }, { $addToSet: { hobbies: "dancing" } } ) // $unset – Remove a field db.users.updateOne( { name: "Suresh" }, { $unset: { orders: "" } } )
✏️ Update Multiple Documents
// Add a field to all active users db.users.updateMany( { isActive: true }, { $set: { membership: "premium" } } ) // Increment age for all users in Hyderabad db.users.updateMany( { city: "Hyderabad" }, { $inc: { age: 1 } } ) // Set default orders for users without orders db.users.updateMany( { orders: { $exists: false } }, { $set: { orders: 0 } } ) // Rename a field db.users.updateMany( {}, { $rename: { "city": "location" } } )
✏️ Replace Document (Complete Replacement)
// Replaces entire document except _id db.users.replaceOne( { name: "Raj" }, { name: "Raj Kumar", email: "rajkumar@example.com", age: 33, city: "Mumbai", isActive: true } )
6. DELETE – Removing Data
🗑️ Delete One Document
// Delete first matching document db.users.deleteOne({ name: "Suresh" }) // Delete by _id db.users.deleteOne({ _id: ObjectId("507f1f77bcf86cd799439011") })
🗑️ Delete Multiple Documents
// Delete all inactive users db.users.deleteMany({ isActive: false }) // Delete users older than 40 db.users.deleteMany({ age: { $gt: 40 } }) // Delete users from specific city db.users.deleteMany({ city: "Pune" }) // Delete ALL documents (be careful!) // db.users.deleteMany({})
🗑️ Drop Collection (Delete All Documents + Structure)
// Delete the entire collection db.users.drop() // Drop database db.dropDatabase()
7. Bulk Write Operations (Multiple Operations at Once)
db.users.bulkWrite([ // Insert a new user { insertOne: { document: { name: "Kiran", email: "kiran@example.com", age: 29, city: "Hyderabad" } } }, // Update an existing user { updateOne: { filter: { name: "Ramesh" }, update: { $set: { age: 27 } } } }, // Delete a user { deleteOne: { filter: { name: "Raj" } } } ])
8. Upsert – Update or Insert (If Not Exists)

Upsert updates a document if it exists, or inserts it if it doesn't. Very useful for avoiding duplicate checks!

// Update user if exists, otherwise create new one db.users.updateOne( { email: "newuser@example.com" }, { $set: { name: "New User", age: 25, city: "Mumbai", isActive: true } }, { upsert: true } ) // Response shows if it was upserted { acknowledged: true, matchedCount: 0, modifiedCount: 0, upsertedId: ObjectId("...") }
9. CRUD in MongoDB Compass (Visual Guide)
CREATE in Compass

• Click "Add Data" → "Insert Document"
• Or "Import File" for bulk insert
• Type JSON directly or use form view

READ in Compass

• Use filter bar to query
• Click "Options" for advanced filters
• Sort dropdown for ordering
• Project fields to show/hide

UPDATE in Compass

• Hover document → click pencil icon
• Modify fields directly
• Or use "Update Many" for bulk changes
• Click "Update" to save

DELETE in Compass

• Hover document → click trash icon
• Confirm deletion
• Or use filter + "Delete Documents"
• Be careful with delete many!

10. Complete Practice Scenario – Blog Platform
// ========== 1. CREATE ========== // Create posts collection db.createCollection("posts") // Insert a blog post db.posts.insertOne({ title: "My First MongoDB Post", content: "This is the content of my blog post...", author: "Ramesh", tags: ["mongodb", "database", "tutorial"], views: 0, likes: 0, publishedAt: new Date() }) // Insert multiple posts db.posts.insertMany([ { title: "Express.js Guide", content: "Express is a Node.js framework...", author: "Priya", tags: ["express", "nodejs"], views: 150, likes: 25, publishedAt: new Date("2024-01-15") }, { title: "React Hooks Deep Dive", content: "Learn all about React Hooks...", author: "Ramesh", tags: ["react", "frontend"], views: 300, likes: 45, publishedAt: new Date("2024-02-01") }, { title: "Mongoose ODM Tutorial", content: "Mongoose makes MongoDB easy...", author: "Suresh", tags: ["mongodb", "mongoose", "nodejs"], views: 89, likes: 12, publishedAt: new Date("2024-02-10") } ]) // ========== 2. READ ========== // Find all posts by Ramesh db.posts.find({ author: "Ramesh" }) // Find posts with >100 views db.posts.find({ views: { $gt: 100 } }) // Find posts with MongoDB tag AND >50 views db.posts.find({ tags: "mongodb", views: { $gt: 50 } }) // Sort by most viewed db.posts.find().sort({ views: -1 }) // Pagination: page 2 with 2 items per page db.posts.find().skip(2).limit(2) // ========== 3. UPDATE ========== // Increment views on a post db.posts.updateOne( { title: "My First MongoDB Post" }, { $inc: { views: 1 } } ) // Add new tag to all posts db.posts.updateMany( {}, { $addToSet: { tags: "2024" } } ) // Update multiple fields db.posts.updateOne( { title: "Express.js Guide" }, { $set: { views: 200, likes: 30 }, $inc: { version: 1 } } ) // ========== 4. DELETE ========== // Delete a specific post db.posts.deleteOne({ title: "Mongoose ODM Tutorial" }) // Delete old posts (before 2024) db.posts.deleteMany({ publishedAt: { $lt: new Date("2024-01-01") } }) // Verify results db.posts.find().pretty()
