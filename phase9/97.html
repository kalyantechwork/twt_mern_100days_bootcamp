6️⃣ Environment Variables — Frontend Config
# Vite requires VITE_ prefix for environment variables
VITE_API_URL=https://your-backend.onrender.com/api

# Use in your code
const API_URL = import.meta.env.VITE_API_URL;

// Example API call
const response = await axios.get(`${API_URL}/products`);
🔑 Pro Tip: Use different API URLs for development vs production: VITE_API_URL=http://localhost:5000/api (local) VITE_API_URL=https://your-backend.onrender.com/api (production)
7️⃣ Connect Frontend to Deployed Backend
// frontend/src/services/api.js

// CORS must be configured on backend to accept frontend origin
// backend/server.js
const allowedOrigins = [
'http://localhost:5173',
'https://your-app.vercel.app',
'https://your-app.netlify.app'
];
8️⃣ SPA Routing Fix — 404 on Refresh
# Vercel: Create vercel.json in root
{
"rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
}

# Netlify: Create netlify.toml in root
[[redirects]]
from = "/*"
to = "/index.html"
status = 200
🚨 Why this is needed: React Router creates client-side routes. Without this fix, refreshing a page like `/about` gives a 404. This redirects all routes to index.html.
9️⃣ Custom Domain — Professional Look
# Vercel: Project → Settings → Domains → Add domain
# Then add CNAME record in your DNS provider:
# Type: CNAME, Name: www, Value: cname.vercel-dns.com

# Netlify: Site settings → Domain management → Add custom domain
# Then add CNAME record: your-domain.netlify.app

⚠️ Common Frontend Deployment Issues
  • ❌ Blank page after deploy? Check dist folder exists, build command correct.
  • ❌ API calls fail? Check VITE_API_URL environment variable in platform.
  • ❌ 404 on refresh? Add SPA routing fix (vercel.json/netlify.toml).
  • ❌ Images not loading? Use absolute paths or import images.
  • ❌ CORS errors? Add frontend URL to backend allowed origins.

💼 Common Interview Questions

❓ How to deploy React app?

Build with npm run build, then deploy to Vercel/Netlify via GitHub.

❓ What is the build command?

Vite: npm run build, output to dist folder.

❓ How to fix 404 on refresh in SPA?

Add redirect rule to always serve index.html (vercel.json).

❓ Vercel vs Netlify?

Both great — Vercel best for Next.js, Netlify for forms/functions.
