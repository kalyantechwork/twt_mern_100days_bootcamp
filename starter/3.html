1️⃣ What is HTML?

HTML = HyperText Markup Language. Skeleton of webpages.

🏠 Analogy: HTML = Walls & rooms (structure) · CSS = Paint & design · JS = Electricity
2️⃣ Your First HTML Skeleton (Live demo)

📄 Basic index.html code

<!DOCTYPE html>
<html>
<head><title>My First Page</title></head>
<body>
  <h1>Welcome to HTML</h1>
  <p>This is my first webpage.</p>
</body></html>

⚠️ Common Mistakes & Solutions
  • ❌ Not closing tags → always use </h1> etc.
  • ❌ Wrong file extension → save as .html
  • ❌ Typing <hl> instead of <h1>
  • ❌ Forgetting alt attribute → always add meaningful alt
