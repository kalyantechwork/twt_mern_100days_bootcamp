⚠️ Common Mistakes & Solutions
  • ❌ Forgetting controls attribute → audio/video won't show controls → always add controls
  • ❌ Missing alt attribute → bad accessibility → always describe image
  • ❌ Using divs for everything → semantic tags improve SEO & screen readers
  • ❌ Wrong iframe URL (embed link) → use embed/ instead of watch?v=
