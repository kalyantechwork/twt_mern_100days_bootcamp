📊 Project 1: Counter App (Increase / Decrease / Reset)

📚 Concepts Used in Counter App

  • Variables (let): let count = 0; stores the current number that changes over time.
  • DOM Selection: document.getElementById() or querySelector() to access HTML elements like the display and buttons.
  • addEventListener: Listens for "click" events on buttons and triggers functions when clicked.
  • textContent: Updates the displayed number dynamically without reloading the page.
  • State Management: Keeping track of count value and updating UI whenever it changes.

🔢 Counter with Step-by-Step Explanation

💡 How it works: The counter starts at 0. When you click the "+" button, JavaScript adds 1 to the count variable and updates the display. The "-" button subtracts 1. The Reset button sets the count back to 0.

// Step 1: Create a variable to store count
let count = 0;

// Step 2: Select the display element and buttons
const display = document.getElementById("counterDisplay");
const incBtn = document.getElementById("incCounter");

// Step 3: Add event listener to increase button
incBtn.addEventListener("click", () => {
count++; // increment count
display.textContent = count; // update UI
});

🧮 Project 2: Calculator (Add, Subtract, Multiply, Divide)

📚 Concepts Used in Calculator

  • Input Values & Type Conversion: .value returns a string from input fields, so we use Number() to convert to numbers for math operations.
  • Arithmetic Operators: +, -, *, / perform calculations on the converted numbers.
  • Conditional Logic: Check division by zero to avoid errors (can't divide by 0 in math).
  • Multiple Event Listeners: Different buttons perform different operations on the same input values.
  • toFixed(): Format decimal results to 2 decimal places for cleaner display.

📟 Calculator with Explanation

💡 Why Number() is important: Without Number(), "10" + "5" becomes "105" (string concatenation). With Number(), it becomes 15 (numeric addition). Always convert input values!

// Step 1: Get input values and convert to numbers
let a = Number(num1.value);
let b = Number(num2.value);

// Step 2: Perform calculation based on button clicked
function add() { return a + b; }
function subtract() { return a - b; }

// Step 3: Display result on the page
resultDisplay.textContent = "Result: " + result;

✅ Project 3: To-Do App (Add & Delete Tasks)

📚 Concepts Used in To-Do App

  • createElement(): Dynamically creates new HTML elements (div, span, button) in memory.
  • appendChild(): Adds created elements to the actual DOM tree so they appear on the page.
  • remove(): Deletes an element from the DOM when the delete button is clicked.
  • Event Listeners on Dynamic Elements: Each delete button gets its own click listener when created.
  • Keyboard Events: keypress event to detect when user presses "Enter" key to add tasks.
  • Input Validation: Check if task is empty before adding to avoid blank tasks.

📝 To-Do List with Explanation

💡 Dynamic Creation: Each task is NOT written in HTML. Instead, JavaScript createElement() creates brand new elements when you add a task. The delete button removes that specific task using remove().

// Step 1: Create new elements dynamically
const li = document.createElement("li");
const span = document.createElement("span");
const deleteBtn = document.createElement("button");

// Step 2: Add content and styling
span.textContent = taskText;
deleteBtn.textContent = "Delete";

// Step 3: Add delete functionality
deleteBtn.addEventListener("click", () => li.remove());

// Step 4: Append to parent
li.appendChild(span);
li.appendChild(deleteBtn);
todoList.appendChild(li);

🎨 Bonus Project: Random Color Generator

📚 Concepts Used in Color Generator

  • Math.random(): Generates a random decimal between 0 and 1 (like 0.345, 0.892).
  • Math.floor(): Rounds down to get a whole integer (removes decimal part).
  • toString(16): Converts number to hexadecimal (base 16) which uses digits 0-9 and letters A-F — perfect for hex color codes.
  • padStart(): Ensures we always have 6 digits for hex color (adds leading zeros if needed).
  • Dynamic Style Change: element.style.backgroundColor to change color instantly.

🌈 Random Hex Color Generator

💡 How hex colors work: #RRGGBB where RR = red, GG = green, BB = blue (each from 00 to ff in hex). 16777215 is the decimal equivalent of ffffff (maximum hex value). Multiplying by Math.random() gives any possible color!

// Generate random hex color: # + 6 random hex digits
const randomNum = Math.floor(Math.random() * 16777215);
const randomColor = `#${randomNum.toString(16).padStart(6, '0')}`;
element.style.backgroundColor = randomColor;

⚠️ Common Beginner Mistakes & Fixes
🚫 Avoid These Errors:
  • ❌ Wrong selector IDs: always match HTML id exactly → use getElementById with correct spelling
  • ❌ Forgetting Number() conversion: "10" + "5" = "105" → fix with Number() or parseFloat()
  • ❌ Not clearing input after adding task: bad UX → set input.value = "" after adding
  • ❌ Script running before DOM loads: place script before closing </body> or use DOMContentLoaded
  • ❌ Creating duplicate IDs: IDs must be unique on a page → use classes for multiple elements
  • ❌ Division by zero: always check if divisor is not zero before dividing
