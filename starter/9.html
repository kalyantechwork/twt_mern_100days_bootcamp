⚠️ Common DOM Mistakes
  • ❌ Script runs before DOM loads → use DOMContentLoaded or place script at bottom.
  • ❌ Wrong selector syntax: querySelector('.myclass') correct, missing dot for class.
  • ❌ Forgetting event.preventDefault() → form reloads page.
  • ❌ Using innerHTML with user input → security risk (XSS).
  • ❌ Not checking if element exists before manipulating → use if(element).
